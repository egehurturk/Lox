# The Lox Language
---
Lox is a C-like programming language.


### Examples:



```
// Your first Lox program
print "Hello, world!";
```

```
// Data types:

// Booleans:
true;
false;

// Numbers:
1234; // an integer
12.34; // a decimal number

// Strings:
"I am a string";
""; // empty string

// Nil
nil;
```
